# 一、图的基本分类


| 类型     | 说明                                         | 重点                    |
| -------- | -------------------------------------------- | ----------------------- |
| 无向图   | 边没有方向，边记作$(u,v)$                    | 看“度数和 = 2 倍边数” |
| 有向图   | 边有方向，边称为弧，记作$\langle u,v\rangle$ | 看“入度、出度”        |
| 连通图   | 无向图中任意两个顶点之间都有路径             | 连通图至少有$n-1$ 条边  |
| 非连通图 | 不是所有顶点都互相可达                       | 会分成多个连通分量      |
| 强连通图 | 有向图中任意两个顶点都能互相到达             | 比弱连通要求更高        |
| 弱连通图 | 有向图忽略方向后是连通图                     | 只考虑是否整体连起来    |

# 二、无向图的核心关系式

## 1. 无向图度数和公式

$$
\sum_{i=1}^{n} d(v_i)=2e
$$

**变量说明：**


| 变量                   | 含义                                     |
| ---------------------- | ---------------------------------------- |
| $n$                    | 无向图中的顶点个数                       |
| $v_i$                  | 第$i$ 个顶点                             |
| $d(v_i)$               | 顶点$v_i$ 的度，也就是与该顶点相连的边数 |
| $e$                    | 无向图中的边数                           |
| $\sum_{i=1}^{n}d(v_i)$ | 所有顶点的度数之和                       |

**解释：**
无向图中，每条边连接两个顶点，所以每条边会贡献 2 个度。

---

## 2. 已知边数求度数和

例如：一个无向图有 23 条边。

$$
2e=2\times 23=46
$$

**变量说明：**


| 变量 | 含义               |
| ---- | ------------------ |
| $e$  | 无向图中的边数     |
| $23$ | 题目给出的边数     |
| $2e$ | 所有顶点的度数之和 |
| $46$ | 所有顶点度数和     |

---

## 3. 奇度顶点个数

无向图所有顶点度数之和为：

$$
2e
$$

**变量说明：**


| 变量 | 含义               |
| ---- | ------------------ |
| $e$  | 无向图中的边数     |
| $2e$ | 所有顶点的度数之和 |

因为 $2e$ 一定是偶数，所以：

> **无向图中奇度顶点的个数一定是偶数。**


| 情况            | 是否可能 |
| --------------- | -------- |
| 有 2 个奇度顶点 | 可能     |
| 有 4 个奇度顶点 | 可能     |
| 有 5 个奇度顶点 | 不可能   |
| 有 7 个奇度顶点 | 不可能   |

---

# 三、连通图与连通分量

## 1. 无向连通图最少边数

如果一个无向图有 $n$ 个顶点，并且它是连通图，那么至少需要：

$$
n-1
$$

条边。

**变量说明：**


| 变量  | 含义                           |
| ----- | ------------------------------ |
| $n$   | 连通图中的顶点个数             |
| $n-1$ | 让$n$ 个顶点连通所需的最少边数 |

所以有：

$$
e\ge n-1
$$

**变量说明：**


| 变量  | 含义                   |
| ----- | ---------------------- |
| $e$   | 无向连通图中的边数     |
| $n$   | 无向连通图中的顶点个数 |
| $n-1$ | 无向连通图的最少边数   |

---

## 2. 无向简单连通图边数范围

$$
n-1 \le e \le \frac{n(n-1)}{2}
$$

**变量说明：**


| 变量               | 含义                               |
| ------------------ | ---------------------------------- |
| $n$                | 无向简单连通图中的顶点个数         |
| $e$                | 无向简单连通图中的边数             |
| $n-1$              | 连通图最少边数，对应树             |
| $\frac{n(n-1)}{2}$ | 无向简单图最多边数，对应无向完全图 |


| 图类型     | 边数                             |
| ---------- | -------------------------------- |
| 最少连通图 | $n-1$，也就是树                  |
| 最多连通图 | $\frac{n(n-1)}{2}$，也就是完全图 |

---

## 3. 连通分量

如果一个无向图不是连通的，它会分成若干个互不相连的部分，每一部分叫一个**连通分量**。


| 概念     | 说明                         |
| -------- | ---------------------------- |
| 连通分量 | 极大连通子图                 |
| 极大     | 不能再加入其他顶点仍保持连通 |
| 连通图   | 只有 1 个连通分量            |
| 非连通图 | 有 2 个或更多连通分量        |

---

# 四、树和生成树

## 1. 树的边数公式

树是一个**连通且无回路的无向图**。

如果树有 $n$ 个顶点，那么一定有：

$$
e=n-1
$$

**变量说明：**


| 变量  | 含义           |
| ----- | -------------- |
| $e$   | 树中的边数     |
| $n$   | 树中的顶点个数 |
| $n-1$ | 树的固定边数   |

---

## 2. 树的度数和公式

因为树有 $n-1$ 条边，所以树中所有顶点的度数和为：

$$
\sum_{i=1}^{n} d(v_i)=2(n-1)
$$

**变量说明：**


| 变量                   | 含义                                     |
| ---------------------- | ---------------------------------------- |
| $n$                    | 树中的顶点个数                           |
| $v_i$                  | 树中的第$i$ 个顶点                       |
| $d(v_i)$               | 顶点$v_i$ 的度                           |
| $\sum_{i=1}^{n}d(v_i)$ | 树中所有顶点的度数之和                   |
| $2(n-1)$               | 树中所有顶点度数之和，因为树有$n-1$ 条边 |

---

## 3. 树的等价条件

对于一个有 $n$ 个顶点的无向图，下面这些说法等价：


| 条件                             | 是否能推出是树 |
| -------------------------------- | -------------- |
| 连通且无回路                     | 是树           |
| 连通且有$n-1$ 条边               | 是树           |
| 无回路且有$n-1$ 条边             | 是树           |
| 任意两个顶点之间有且只有一条路径 | 是树           |
| 删除任意一条边都会不连通         | 是树           |
| 加入任意一条新边都会产生回路     | 是树           |

其中：

$$
n-1
$$

**变量说明：**


| 变量  | 含义               |
| ----- | ------------------ |
| $n$   | 图中的顶点个数     |
| $n-1$ | 树中必须具有的边数 |

---

## 4. 生成树边数公式

生成树是连通图的一个子图，它包含原图所有顶点，并且用最少边保持连通。

$$
\text{生成树边数}=n-1
$$

**变量说明：**


| 变量  | 含义                                           |
| ----- | ---------------------------------------------- |
| $n$   | 原连通图中的顶点个数，也等于生成树中的顶点个数 |
| $n-1$ | 生成树中的边数                                 |

---

## 5. 生成树的常考判断


| 说法                                 | 判断 |
| ------------------------------------ | ---- |
| 一个连通图一定有生成树               | 对   |
| 生成树包含原图所有顶点               | 对   |
| 生成树包含原图所有边                 | 错   |
| 生成树有$n-1$ 条边                   | 对   |
| 生成树不一定唯一                     | 对   |
| 若图本身是一棵树，则生成树就是它自己 | 对   |

---

# 五、森林和生成森林

森林就是若干棵树的集合，也就是**无回路的无向图**。

## 1. 森林边数公式

如果森林有 $n$ 个顶点，$c$ 个连通分量，那么边数为：

$$
e=n-c
$$

**变量说明：**


| 变量  | 含义                                 |
| ----- | ------------------------------------ |
| $e$   | 森林中的边数                         |
| $n$   | 森林中的顶点个数                     |
| $c$   | 森林中的连通分量个数，也就是树的棵数 |
| $n-c$ | 森林的边数                           |

---

## 2. 森林连通分量公式

由森林边数公式可以变形得到：

$$
c=n-e
$$

**变量说明：**


| 变量  | 含义                               |
| ----- | ---------------------------------- |
| $c$   | 森林中的连通分量个数               |
| $n$   | 森林中的顶点个数                   |
| $e$   | 森林中的边数                       |
| $n-e$ | 根据顶点数和边数求出的连通分量个数 |

---

## 3. 树是特殊森林

当森林只有 1 个连通分量时，它就是一棵树。

$$
c=1
$$

**变量说明：**


| 变量 | 含义                               |
| ---- | ---------------------------------- |
| $c$  | 连通分量个数                       |
| $1$  | 表示只有一个连通分量，即整张图连通 |

此时：

$$
e=n-1
$$

**变量说明：**


| 变量  | 含义           |
| ----- | -------------- |
| $e$   | 树中的边数     |
| $n$   | 树中的顶点个数 |
| $n-1$ | 树的边数       |

---

# 六、完全图

## 1. 无向完全图边数公式

无向完全图记作：

$$
K_n
$$

**变量说明：**


| 变量  | 含义                     |
| ----- | ------------------------ |
| $K_n$ | 有$n$ 个顶点的无向完全图 |
| $n$   | 无向完全图中的顶点个数   |

无向完全图中，任意两个不同顶点之间都有一条边。

$$
e=\frac{n(n-1)}{2}
$$

**变量说明：**


| 变量               | 含义                                                             |
| ------------------ | ---------------------------------------------------------------- |
| $e$                | 无向完全图中的边数                                               |
| $n$                | 无向完全图中的顶点个数                                           |
| $n-1$              | 每个顶点可以和除自己外的其他$n-1$ 个顶点相连                     |
| $\frac{n(n-1)}{2}$ | 无向完全图的边数，因为每条边会被两个端点重复计算一次，所以除以 2 |

---

## 2. 无向完全图顶点度数

$$
d(v)=n-1
$$

**变量说明：**


| 变量   | 含义                         |
| ------ | ---------------------------- |
| $v$    | 无向完全图中的任意一个顶点   |
| $d(v)$ | 顶点$v$ 的度                 |
| $n$    | 无向完全图中的顶点个数       |
| $n-1$  | 顶点$v$ 能连接的其他顶点个数 |

---

## 3. 有向完全图弧数公式

有向完全图中，任意两个不同顶点之间都有两个方向的弧。

$$
e=n(n-1)
$$

**变量说明：**


| 变量     | 含义                                       |
| -------- | ------------------------------------------ |
| $e$      | 有向完全图中的弧数                         |
| $n$      | 有向完全图中的顶点个数                     |
| $n-1$    | 每个顶点可以指向除自己外的其他$n-1$ 个顶点 |
| $n(n-1)$ | 有向完全图的弧数                           |

---

## 4. 有向完全图入度和出度

$$
ID(v)=n-1
$$

**变量说明：**


| 变量    | 含义                             |
| ------- | -------------------------------- |
| $v$     | 有向完全图中的任意一个顶点       |
| $ID(v)$ | 顶点$v$ 的入度                   |
| $n$     | 有向完全图中的顶点个数           |
| $n-1$   | 除$v$ 外，其他顶点都可以指向 $v$ |

$$
OD(v)=n-1
$$

**变量说明：**


| 变量    | 含义                                 |
| ------- | ------------------------------------ |
| $v$     | 有向完全图中的任意一个顶点           |
| $OD(v)$ | 顶点$v$ 的出度                       |
| $n$     | 有向完全图中的顶点个数               |
| $n-1$   | 顶点$v$ 可以指向除自己外的其他顶点数 |

$$
ID(v)+OD(v)=2(n-1)
$$

**变量说明：**


| 变量     | 含义                       |
| -------- | -------------------------- |
| $v$      | 有向完全图中的任意一个顶点 |
| $ID(v)$  | 顶点$v$ 的入度             |
| $OD(v)$  | 顶点$v$ 的出度             |
| $n$      | 有向完全图中的顶点个数     |
| $2(n-1)$ | 顶点$v$ 的总度数           |

---

# 七、有向图的入度和出度

## 1. 弧的表示

有向图中，一条弧可以写成：

$$
u\to v
$$

**变量说明：**


| 变量     | 含义                            |
| -------- | ------------------------------- |
| $u$      | 弧的起点，也叫弧尾              |
| $v$      | 弧的终点，也叫弧头              |
| $u\to v$ | 从顶点$u$ 指向顶点 $v$ 的有向边 |

---

## 2. 有向图入度和公式

$$
\sum_{i=1}^{n} ID(v_i)=e
$$

**变量说明：**


| 变量                    | 含义               |
| ----------------------- | ------------------ |
| $n$                     | 有向图中的顶点个数 |
| $v_i$                   | 第$i$ 个顶点       |
| $ID(v_i)$               | 顶点$v_i$ 的入度   |
| $e$                     | 有向图中的弧数     |
| $\sum_{i=1}^{n}ID(v_i)$ | 所有顶点入度之和   |

---

## 3. 有向图出度和公式

$$
\sum_{i=1}^{n} OD(v_i)=e
$$

**变量说明：**


| 变量                    | 含义               |
| ----------------------- | ------------------ |
| $n$                     | 有向图中的顶点个数 |
| $v_i$                   | 第$i$ 个顶点       |
| $OD(v_i)$               | 顶点$v_i$ 的出度   |
| $e$                     | 有向图中的弧数     |
| $\sum_{i=1}^{n}OD(v_i)$ | 所有顶点出度之和   |

---

## 4. 入度和等于出度和

$$
\sum_{i=1}^{n} ID(v_i)=\sum_{i=1}^{n} OD(v_i)=e
$$

**变量说明：**


| 变量           | 含义               |
| -------------- | ------------------ |
| $n$            | 有向图中的顶点个数 |
| $v_i$          | 第$i$ 个顶点       |
| $ID(v_i)$      | 顶点$v_i$ 的入度   |
| $OD(v_i)$      | 顶点$v_i$ 的出度   |
| $e$            | 有向图中的弧数     |
| $\sum ID(v_i)$ | 所有顶点入度之和   |
| $\sum OD(v_i)$ | 所有顶点出度之和   |

---

## 5. 有向图总度数和

$$
\sum_{i=1}^{n} [ID(v_i)+OD(v_i)]=2e
$$

**变量说明：**


| 变量      | 含义               |
| --------- | ------------------ |
| $n$       | 有向图中的顶点个数 |
| $v_i$     | 第$i$ 个顶点       |
| $ID(v_i)$ | 顶点$v_i$ 的入度   |
| $OD(v_i)$ | 顶点$v_i$ 的出度   |
| $e$       | 有向图中的弧数     |
| $2e$      | 所有顶点总度数之和 |

---

# 八、强连通有向图

## 1. 强连通的可达关系

有向图中，任意两个顶点 $u,v$，都满足：

$$
u\to v
$$

**变量说明：**


| 变量     | 含义                                 |
| -------- | ------------------------------------ |
| $u$      | 有向图中的任意一个顶点               |
| $v$      | 有向图中另一个任意顶点               |
| $u\to v$ | 从顶点$u$ 可以沿有向路径到达顶点 $v$ |

并且：

$$
v\to u
$$

**变量说明：**


| 变量     | 含义                                 |
| -------- | ------------------------------------ |
| $v$      | 有向图中的任意一个顶点               |
| $u$      | 有向图中另一个任意顶点               |
| $v\to u$ | 从顶点$v$ 可以沿有向路径到达顶点 $u$ |

这时称该有向图为**强连通图**。

---

## 2. 强连通有向图最少弧数

如果有向强连通图有 $n$ 个顶点，并且 $n\ge 2$，则最少需要：

$$
n
$$

条弧。

**变量说明：**


| 变量     | 含义                                       |
| -------- | ------------------------------------------ |
| $n$      | 强连通有向图中的顶点个数                   |
| $n\ge 2$ | 表示至少有两个顶点                         |
| $n$ 条弧 | 构成一个包含所有顶点的有向环所需的最少弧数 |

---

## 3. 最少强连通结构：有向环

$$
v_1\to v_2\to v_3\to \cdots \to v_n\to v_1
$$

**变量说明：**


| 变量                 | 含义                                   |
| -------------------- | -------------------------------------- |
| $v_1,v_2,\ldots,v_n$ | 强连通有向图中的$n$ 个顶点             |
| $\to$                | 有向弧或有向路径方向                   |
| $v_n\to v_1$         | 最后一个顶点回到第一个顶点，形成有向环 |

---

## 4. 强连通有向图边数范围

$$
n \le e \le n(n-1)
$$

**变量说明：**


| 变量     | 含义                                   |
| -------- | -------------------------------------- |
| $n$      | 强连通有向图中的顶点个数               |
| $e$      | 强连通有向图中的弧数                   |
| $n$      | 强连通有向图的最少弧数，对应一个有向环 |
| $n(n-1)$ | 强连通有向图的最多弧数，对应有向完全图 |

---

## 5. 强连通、弱连通关系


| 关系                       | 是否成立 |
| -------------------------- | -------- |
| 强连通一定弱连通           | 成立     |
| 弱连通一定强连通           | 不成立   |
| 强连通图忽略方向后一定连通 | 成立     |
| 有向图连通不一定强连通     | 成立     |

---

# 九、子图相关概念

设原图为：

$$
G=(V,E)
$$

**变量说明：**


| 变量 | 含义                 |
| ---- | -------------------- |
| $G$  | 原图                 |
| $V$  | 原图的顶点集合       |
| $E$  | 原图的边集合或弧集合 |

子图为：

$$
G'=(V',E')
$$

**变量说明：**


| 变量 | 含义                 |
| ---- | -------------------- |
| $G'$ | 原图$G$ 的子图       |
| $V'$ | 子图的顶点集合       |
| $E'$ | 子图的边集合或弧集合 |

子图必须满足：

$$
V'\subseteq V
$$

**变量说明：**


| 变量        | 含义                 |
| ----------- | -------------------- |
| $V'$        | 子图的顶点集合       |
| $V$         | 原图的顶点集合       |
| $\subseteq$ | 表示“是……的子集” |

并且：

$$
E'\subseteq E
$$

**变量说明：**


| 变量        | 含义                 |
| ----------- | -------------------- |
| $E'$        | 子图的边集合或弧集合 |
| $E$         | 原图的边集合或弧集合 |
| $\subseteq$ | 表示“是……的子集” |

---

## 子图分类


| 类型     | 说明                           |
| -------- | ------------------------------ |
| 子图     | 顶点和边都是原图的一部分       |
| 生成子图 | 包含原图所有顶点，只删边不删点 |
| 生成树   | 连通图的极小连通生成子图       |
| 连通分量 | 极大连通子图                   |

---

# 十、边数与连通分量的常考关系

设无向图有 $n$ 个顶点，$e$ 条边，$c$ 个连通分量。

## 1. 最少连通分量下界

一开始 $n$ 个顶点，如果没有边，就是 $n$ 个连通分量。
每增加一条边，最多可以让连通分量减少 1。

所以：

$$
c\ge n-e
$$

**变量说明：**


| 变量  | 含义                   |
| ----- | ---------------------- |
| $c$   | 无向图中的连通分量个数 |
| $n$   | 无向图中的顶点个数     |
| $e$   | 无向图中的边数         |
| $n-e$ | 连通分量数量的一个下界 |

---

## 2. 最少连通分量公式

如果边足够多，最少可以连成一个连通图。

$$
c_{\min}=1
$$

**变量说明：**


| 变量       | 含义                             |
| ---------- | -------------------------------- |
| $c_{\min}$ | 最少连通分量个数                 |
| $1$        | 表示整张图连通，只有一个连通分量 |

综合考虑：

$$
c_{\min}=\max(1,n-e)
$$

**变量说明：**


| 变量          | 含义                                            |
| ------------- | ----------------------------------------------- |
| $c_{\min}$    | 最少连通分量个数                                |
| $n$           | 无向图中的顶点个数                              |
| $e$           | 无向图中的边数                                  |
| $\max(1,n-e)$ | 在$1$ 和 $n-e$ 中取较大值，保证连通分量至少为 1 |

---

## 3. 最多连通分量问题的核心不等式

想让连通分量尽可能多，就要把边尽量集中放在少数顶点中，让其他顶点孤立。

$$
\frac{s(s-1)}{2}\ge e
$$

**变量说明：**


| 变量               | 含义                                               |
| ------------------ | -------------------------------------------------- |
| $s$                | 用来集中容纳$e$ 条边所需的最少顶点数               |
| $e$                | 无向图中的边数                                     |
| $\frac{s(s-1)}{2}$ | $s$ 个顶点的无向完全图最多能容纳的边数             |
| $\ge$              | 表示这些顶点最多容纳的边数必须不少于题目给出的边数 |

---

## 4. 示例：21 条边需要几个顶点容纳

$$
\frac{s(s-1)}{2}\ge 21
$$

**变量说明：**


| 变量               | 含义                         |
| ------------------ | ---------------------------- |
| $s$                | 容纳 21 条边所需的最少顶点数 |
| $21$               | 题目给出的边数               |
| $\frac{s(s-1)}{2}$ | $s$ 个顶点最多能形成的边数   |

当 $s=7$ 时：

$$
\frac{7\times 6}{2}=21
$$

**变量说明：**


| 变量 | 含义                                   |
| ---- | -------------------------------------- |
| $7$  | 用来集中放置这些边的顶点数             |
| $6$  | 除自身外，每个顶点最多连接的其他顶点数 |
| $21$ | 7 个顶点构成无向完全图时的边数         |

如果总共有 51 个顶点，那么剩下孤立顶点数为：

$$
51-7=44
$$

**变量说明：**


| 变量 | 含义                     |
| ---- | ------------------------ |
| $51$ | 图中的总顶点数           |
| $7$  | 用来容纳 21 条边的顶点数 |
| $44$ | 剩下的孤立顶点数         |

总连通分量数为：

$$
44+1=45
$$

**变量说明：**


| 变量 | 含义                                       |
| ---- | ------------------------------------------ |
| $44$ | 孤立顶点数量，每个孤立顶点都是一个连通分量 |
| $1$  | 由 7 个顶点和 21 条边组成的那个连通分量    |
| $45$ | 最多连通分量个数                           |

# 十一、判断图是否一定连通

## 1. 有 $n-1$ 条边，不一定连通

很多人会误以为：

$$
e=n-1
$$

**变量说明：**


| 变量  | 含义                           |
| ----- | ------------------------------ |
| $e$   | 无向图中的边数                 |
| $n$   | 无向图中的顶点个数             |
| $n-1$ | 树的边数，也是连通图的最少边数 |

就一定是树，这是错的。
必须还要满足“连通”。

---

## 2. 例子：4 个顶点 3 条边不一定连通

$$
n=4,\quad e=3=n-1
$$

**变量说明：**


| 变量  | 含义              |
| ----- | ----------------- |
| $n$   | 图中的顶点个数    |
| $4$   | 题目给出的顶点数  |
| $e$   | 图中的边数        |
| $3$   | 题目给出的边数    |
| $n-1$ | $4-1=3$，树的边数 |

虽然满足 $e=n-1$，但如果 3 个顶点形成三角形，另一个顶点孤立，它不是连通图，也不是树。

---

## 3. 连通且有 $n-1$ 条边，一定是树

$$
\text{连通} + e=n-1 \Rightarrow \text{树}
$$

**变量说明：**


| 变量          | 含义               |
| ------------- | ------------------ |
| $e$           | 无向图中的边数     |
| $n$           | 无向图中的顶点个数 |
| $n-1$         | 树的边数           |
| $\Rightarrow$ | 表示“可以推出”   |

或者：

$$
\text{无回路} + e=n-1 \Rightarrow \text{树}
$$

**变量说明：**


| 变量          | 含义               |
| ------------- | ------------------ |
| $e$           | 无向图中的边数     |
| $n$           | 无向图中的顶点个数 |
| $n-1$         | 树的边数           |
| $\Rightarrow$ | 表示“可以推出”   |

---

## 4. 边数少于 $n-1$，一定不连通

$$
e<n-1
$$

**变量说明：**


| 变量  | 含义                 |
| ----- | -------------------- |
| $e$   | 无向图中的边数       |
| $n$   | 无向图中的顶点个数   |
| $n-1$ | 连通图至少需要的边数 |
| $<$   | 表示“少于”         |

如果边数少于 $n-1$，那么图一定不连通。

---

## 5. 边数足够多时一定连通

一个 $n$ 个顶点的无向简单图，如果不连通，最多可以让 $n-1$ 个顶点构成完全图，另一个顶点孤立。

$$
\frac{(n-1)(n-2)}{2}
$$

**变量说明：**


| 变量                   | 含义                                             |
| ---------------------- | ------------------------------------------------ |
| $n$                    | 无向简单图中的顶点个数                           |
| $n-1$                  | 除孤立点外，其余顶点个数                         |
| $n-2$                  | 在这$n-1$ 个顶点中，每个顶点最多连接的其他顶点数 |
| $\frac{(n-1)(n-2)}{2}$ | $n$ 个顶点的非连通简单图最多边数                 |

所以如果：

$$
e>\frac{(n-1)(n-2)}{2}
$$

**变量说明：**


| 变量                   | 含义                           |
| ---------------------- | ------------------------------ |
| $e$                    | 无向简单图中的边数             |
| $n$                    | 无向简单图中的顶点个数         |
| $\frac{(n-1)(n-2)}{2}$ | 非连通简单图最多可能拥有的边数 |
| $>$                    | 表示“多于”                   |

那么图一定连通。

---

# 十二、回路与桥

## 1. 回路上的边不是桥

如果一条边在某个回路上，那么删除它之后，图仍然连通。


| 边的类型   | 删除后是否连通 |
| ---------- | -------------- |
| 回路上的边 | 仍然连通       |
| 桥         | 会变成不连通   |

---

## 2. 桥的定义

如果删除某条边后，图的连通分量增加，那么这条边就是桥，也叫割边。

树中的每一条边都是桥。

---

# 十三、邻接矩阵相关知识

## 1. 无向图邻接矩阵对称

$$
A[i][j]=A[j][i]
$$

**变量说明：**


| 变量      | 含义                            |
| --------- | ------------------------------- |
| $A$       | 图的邻接矩阵                    |
| $i,j$     | 顶点编号                        |
| $A[i][j]$ | 顶点$v_i$ 到顶点 $v_j$ 是否有边 |
| $A[j][i]$ | 顶点$v_j$ 到顶点 $v_i$ 是否有边 |

无向图没有方向，所以邻接矩阵关于主对角线对称。

---

## 2. 无向图邻接矩阵求度

$$
d(v_i)=\sum_{j=1}^{n} A[i][j]
$$

**变量说明：**


| 变量                    | 含义                                |
| ----------------------- | ----------------------------------- |
| $d(v_i)$                | 顶点$v_i$ 的度                      |
| $v_i$                   | 第$i$ 个顶点                        |
| $A$                     | 无向图的邻接矩阵                    |
| $A[i][j]$               | 顶点$v_i$ 和顶点 $v_j$ 之间是否有边 |
| $n$                     | 图中的顶点个数                      |
| $\sum_{j=1}^{n}A[i][j]$ | 邻接矩阵第$i$ 行元素之和            |

---

## 3. 有向图邻接矩阵求出度

$$
OD(v_i)=\sum_{j=1}^{n} A[i][j]
$$

**变量说明：**


| 变量                    | 含义                                    |
| ----------------------- | --------------------------------------- |
| $OD(v_i)$               | 顶点$v_i$ 的出度                        |
| $v_i$                   | 第$i$ 个顶点                            |
| $A$                     | 有向图的邻接矩阵                        |
| $A[i][j]$               | 从顶点$v_i$ 指向顶点 $v_j$ 的弧是否存在 |
| $n$                     | 图中的顶点个数                          |
| $\sum_{j=1}^{n}A[i][j]$ | 邻接矩阵第$i$ 行元素之和                |

---

## 4. 有向图邻接矩阵求入度

$$
ID(v_i)=\sum_{j=1}^{n} A[j][i]
$$

**变量说明：**


| 变量                    | 含义                                    |
| ----------------------- | --------------------------------------- |
| $ID(v_i)$               | 顶点$v_i$ 的入度                        |
| $v_i$                   | 第$i$ 个顶点                            |
| $A$                     | 有向图的邻接矩阵                        |
| $A[j][i]$               | 从顶点$v_j$ 指向顶点 $v_i$ 的弧是否存在 |
| $n$                     | 图中的顶点个数                          |
| $\sum_{j=1}^{n}A[j][i]$ | 邻接矩阵第$i$ 列元素之和                |

---

# 十五、考试易错点总结


| 易错说法                       | 正确理解                 |
| ------------------------------ | ------------------------ |
| 有$n-1$ 条边的图一定是树       | 错，还要连通             |
| 连通图一定有$n-1$ 条边         | 错，连通图至少$n-1$ 条边 |
| 树一定有$n-1$ 条边             | 对                       |
| 有向图弱连通就是强连通         | 错                       |
| 强连通一定弱连通               | 对                       |
| 无向图所有度数之和等于边数     | 错，是边数的 2 倍        |
| 有向图入度和等于出度和         | 对                       |
| 删除回路上的边会破坏连通性     | 错，仍连通               |
| 树中删除任意一条边都会不连通   | 对                       |
| 树中增加任意一条边都会产生回路 | 对                       |
| 连通分量是普通连通子图         | 错，是极大连通子图       |

# 十六、做这类题的思路

## 第一步：看是无向图还是有向图

无向图优先用：

$$
\sum d(v)=2e
$$

**变量说明：**


| 变量   | 含义           |
| ------ | -------------- |
| $d(v)$ | 顶点$v$ 的度   |
| $e$    | 无向图中的边数 |

有向图优先用：

$$
\sum ID(v)=\sum OD(v)=e
$$

**变量说明：**


| 变量    | 含义           |
| ------- | -------------- |
| $ID(v)$ | 顶点$v$ 的入度 |
| $OD(v)$ | 顶点$v$ 的出度 |
| $e$     | 有向图中的弧数 |

---

## 第二步：看是否连通

如果题目说连通图，马上想到：

$$
e\ge n-1
$$

**变量说明：**


| 变量 | 含义               |
| ---- | ------------------ |
| $e$  | 连通图中的边数     |
| $n$  | 连通图中的顶点个数 |

如果说树，马上想到：

$$
e=n-1
$$

**变量说明：**


| 变量 | 含义           |
| ---- | -------------- |
| $e$  | 树中的边数     |
| $n$  | 树中的顶点个数 |

如果说森林，马上想到：

$$
e=n-c
$$

**变量说明：**


| 变量 | 含义                 |
| ---- | -------------------- |
| $e$  | 森林中的边数         |
| $n$  | 森林中的顶点个数     |
| $c$  | 森林中的连通分量个数 |

---

## 第三步：看是否完全图

无向完全图：

$$
e=\frac{n(n-1)}{2}
$$

**变量说明：**


| 变量 | 含义                   |
| ---- | ---------------------- |
| $e$  | 无向完全图中的边数     |
| $n$  | 无向完全图中的顶点个数 |

有向完全图：

$$
e=n(n-1)
$$

**变量说明：**


| 变量 | 含义                   |
| ---- | ---------------------- |
| $e$  | 有向完全图中的弧数     |
| $n$  | 有向完全图中的顶点个数 |

---

## 第四步：看是否强连通

强连通有向图要求任意两点互相可达。

最少弧数：

$$
e=n
$$

**变量说明：**


| 变量 | 含义                                  |
| ---- | ------------------------------------- |
| $e$  | 强连通有向图中的弧数                  |
| $n$  | 强连通有向图中的顶点个数，要求$n\ge2$ |

---

# 一、图的基本分类


| 类型     | 说明                                           | 重点                     |
| -------- | ---------------------------------------------- | ------------------------ |
| 无向图   | 边没有方向，边记作\((u,v)\)                    | 看“度数和 = 2 倍边数”  |
| 有向图   | 边有方向，边称为弧，记作\(\langle u,v\rangle\) | 看“入度、出度”         |
| 连通图   | 无向图中任意两个顶点之间都有路径               | 连通图至少有\(n-1\) 条边 |
| 非连通图 | 不是所有顶点都互相可达                         | 会分成多个连通分量       |
| 强连通图 | 有向图中任意两个顶点都能互相到达               | 比弱连通要求更高         |
| 弱连通图 | 有向图忽略方向后是连通图                       | 只考虑是否整体连起来     |

# 二、无向图的核心关系式

## 1. 无向图度数和公式

\[
\sum_{i=1}^{n} d(v_i)=2e
\]

**变量说明：**


| 变量                     | 含义                                       |
| ------------------------ | ------------------------------------------ |
| \(n\)                    | 无向图中的顶点个数                         |
| \(v_i\)                  | 第\(i\) 个顶点                             |
| \(d(v_i)\)               | 顶点\(v_i\) 的度，也就是与该顶点相连的边数 |
| \(e\)                    | 无向图中的边数                             |
| \(\sum_{i=1}^{n}d(v_i)\) | 所有顶点的度数之和                         |

**解释：**
无向图中，每条边连接两个顶点，所以每条边会贡献 2 个度。

---

## 2. 已知边数求度数和

例如：一个无向图有 23 条边。

\[
2e=2\times 23=46
\]

**变量说明：**


| 变量   | 含义               |
| ------ | ------------------ |
| \(e\)  | 无向图中的边数     |
| \(23\) | 题目给出的边数     |
| \(2e\) | 所有顶点的度数之和 |
| \(46\) | 所有顶点度数和     |

---

## 3. 奇度顶点个数

无向图所有顶点度数之和为：

\[
2e
\]

**变量说明：**


| 变量   | 含义               |
| ------ | ------------------ |
| \(e\)  | 无向图中的边数     |
| \(2e\) | 所有顶点的度数之和 |

因为 \(2e\) 一定是偶数，所以：

> **无向图中奇度顶点的个数一定是偶数。**


| 情况            | 是否可能 |
| --------------- | -------- |
| 有 2 个奇度顶点 | 可能     |
| 有 4 个奇度顶点 | 可能     |
| 有 5 个奇度顶点 | 不可能   |
| 有 7 个奇度顶点 | 不可能   |

---

# 三、连通图与连通分量

## 1. 无向连通图最少边数

如果一个无向图有 \(n\) 个顶点，并且它是连通图，那么至少需要：

\[
n-1
\]

条边。

**变量说明：**


| 变量    | 含义                             |
| ------- | -------------------------------- |
| \(n\)   | 连通图中的顶点个数               |
| \(n-1\) | 让\(n\) 个顶点连通所需的最少边数 |

所以有：

\[
e\ge n-1
\]

**变量说明：**


| 变量    | 含义                   |
| ------- | ---------------------- |
| \(e\)   | 无向连通图中的边数     |
| \(n\)   | 无向连通图中的顶点个数 |
| \(n-1\) | 无向连通图的最少边数   |

---

## 2. 无向简单连通图边数范围

\[
n-1 \le e \le \frac{n(n-1)}{2}
\]

**变量说明：**


| 变量                 | 含义                               |
| -------------------- | ---------------------------------- |
| \(n\)                | 无向简单连通图中的顶点个数         |
| \(e\)                | 无向简单连通图中的边数             |
| \(n-1\)              | 连通图最少边数，对应树             |
| \(\frac{n(n-1)}{2}\) | 无向简单图最多边数，对应无向完全图 |


| 图类型     | 边数                               |
| ---------- | ---------------------------------- |
| 最少连通图 | \(n-1\)，也就是树                  |
| 最多连通图 | \(\frac{n(n-1)}{2}\)，也就是完全图 |

---

## 3. 连通分量

如果一个无向图不是连通的，它会分成若干个互不相连的部分，每一部分叫一个**连通分量**。


| 概念     | 说明                         |
| -------- | ---------------------------- |
| 连通分量 | 极大连通子图                 |
| 极大     | 不能再加入其他顶点仍保持连通 |
| 连通图   | 只有 1 个连通分量            |
| 非连通图 | 有 2 个或更多连通分量        |

---

# 四、树和生成树

## 1. 树的边数公式

树是一个**连通且无回路的无向图**。

如果树有 \(n\) 个顶点，那么一定有：

\[
e=n-1
\]

**变量说明：**


| 变量    | 含义           |
| ------- | -------------- |
| \(e\)   | 树中的边数     |
| \(n\)   | 树中的顶点个数 |
| \(n-1\) | 树的固定边数   |

---

## 2. 树的度数和公式

因为树有 \(n-1\) 条边，所以树中所有顶点的度数和为：

\[
\sum_{i=1}^{n} d(v_i)=2(n-1)
\]

**变量说明：**


| 变量                     | 含义                                       |
| ------------------------ | ------------------------------------------ |
| \(n\)                    | 树中的顶点个数                             |
| \(v_i\)                  | 树中的第\(i\) 个顶点                       |
| \(d(v_i)\)               | 顶点\(v_i\) 的度                           |
| \(\sum_{i=1}^{n}d(v_i)\) | 树中所有顶点的度数之和                     |
| \(2(n-1)\)               | 树中所有顶点度数之和，因为树有\(n-1\) 条边 |

---

## 3. 树的等价条件

对于一个有 \(n\) 个顶点的无向图，下面这些说法等价：


| 条件                             | 是否能推出是树 |
| -------------------------------- | -------------- |
| 连通且无回路                     | 是树           |
| 连通且有\(n-1\) 条边             | 是树           |
| 无回路且有\(n-1\) 条边           | 是树           |
| 任意两个顶点之间有且只有一条路径 | 是树           |
| 删除任意一条边都会不连通         | 是树           |
| 加入任意一条新边都会产生回路     | 是树           |

其中：

\[
n-1
\]

**变量说明：**


| 变量    | 含义               |
| ------- | ------------------ |
| \(n\)   | 图中的顶点个数     |
| \(n-1\) | 树中必须具有的边数 |

---

## 4. 生成树边数公式

生成树是连通图的一个子图，它包含原图所有顶点，并且用最少边保持连通。

\[
\text{生成树边数}=n-1
\]

**变量说明：**


| 变量    | 含义                                           |
| ------- | ---------------------------------------------- |
| \(n\)   | 原连通图中的顶点个数，也等于生成树中的顶点个数 |
| \(n-1\) | 生成树中的边数                                 |

---

## 5. 生成树的常考判断


| 说法                                 | 判断 |
| ------------------------------------ | ---- |
| 一个连通图一定有生成树               | 对   |
| 生成树包含原图所有顶点               | 对   |
| 生成树包含原图所有边                 | 错   |
| 生成树有\(n-1\) 条边                 | 对   |
| 生成树不一定唯一                     | 对   |
| 若图本身是一棵树，则生成树就是它自己 | 对   |

---

# 五、森林和生成森林

森林就是若干棵树的集合，也就是**无回路的无向图**。

## 1. 森林边数公式

如果森林有 \(n\) 个顶点，\(c\) 个连通分量，那么边数为：

\[
e=n-c
\]

**变量说明：**


| 变量    | 含义                                 |
| ------- | ------------------------------------ |
| \(e\)   | 森林中的边数                         |
| \(n\)   | 森林中的顶点个数                     |
| \(c\)   | 森林中的连通分量个数，也就是树的棵数 |
| \(n-c\) | 森林的边数                           |

---

## 2. 森林连通分量公式

由森林边数公式可以变形得到：

\[
c=n-e
\]

**变量说明：**


| 变量    | 含义                               |
| ------- | ---------------------------------- |
| \(c\)   | 森林中的连通分量个数               |
| \(n\)   | 森林中的顶点个数                   |
| \(e\)   | 森林中的边数                       |
| \(n-e\) | 根据顶点数和边数求出的连通分量个数 |

---

## 3. 树是特殊森林

当森林只有 1 个连通分量时，它就是一棵树。

\[
c=1
\]

**变量说明：**


| 变量  | 含义                               |
| ----- | ---------------------------------- |
| \(c\) | 连通分量个数                       |
| \(1\) | 表示只有一个连通分量，即整张图连通 |

此时：

\[
e=n-1
\]

**变量说明：**


| 变量    | 含义           |
| ------- | -------------- |
| \(e\)   | 树中的边数     |
| \(n\)   | 树中的顶点个数 |
| \(n-1\) | 树的边数       |

---

# 六、完全图

## 1. 无向完全图边数公式

无向完全图记作：

\[
K_n
\]

**变量说明：**


| 变量    | 含义                       |
| ------- | -------------------------- |
| \(K_n\) | 有\(n\) 个顶点的无向完全图 |
| \(n\)   | 无向完全图中的顶点个数     |

无向完全图中，任意两个不同顶点之间都有一条边。

\[
e=\frac{n(n-1)}{2}
\]

**变量说明：**


| 变量                 | 含义                                                             |
| -------------------- | ---------------------------------------------------------------- |
| \(e\)                | 无向完全图中的边数                                               |
| \(n\)                | 无向完全图中的顶点个数                                           |
| \(n-1\)              | 每个顶点可以和除自己外的其他\(n-1\) 个顶点相连                   |
| \(\frac{n(n-1)}{2}\) | 无向完全图的边数，因为每条边会被两个端点重复计算一次，所以除以 2 |

---

## 2. 无向完全图顶点度数

\[
d(v)=n-1
\]

**变量说明：**


| 变量     | 含义                           |
| -------- | ------------------------------ |
| \(v\)    | 无向完全图中的任意一个顶点     |
| \(d(v)\) | 顶点\(v\) 的度                 |
| \(n\)    | 无向完全图中的顶点个数         |
| \(n-1\)  | 顶点\(v\) 能连接的其他顶点个数 |

---

## 3. 有向完全图弧数公式

有向完全图中，任意两个不同顶点之间都有两个方向的弧。

\[
e=n(n-1)
\]

**变量说明：**


| 变量       | 含义                                         |
| ---------- | -------------------------------------------- |
| \(e\)      | 有向完全图中的弧数                           |
| \(n\)      | 有向完全图中的顶点个数                       |
| \(n-1\)    | 每个顶点可以指向除自己外的其他\(n-1\) 个顶点 |
| \(n(n-1)\) | 有向完全图的弧数                             |

---

## 4. 有向完全图入度和出度

\[
ID(v)=n-1
\]

**变量说明：**


| 变量      | 含义                                 |
| --------- | ------------------------------------ |
| \(v\)     | 有向完全图中的任意一个顶点           |
| \(ID(v)\) | 顶点\(v\) 的入度                     |
| \(n\)     | 有向完全图中的顶点个数               |
| \(n-1\)   | 除\(v\) 外，其他顶点都可以指向 \(v\) |

\[
OD(v)=n-1
\]

**变量说明：**


| 变量      | 含义                                   |
| --------- | -------------------------------------- |
| \(v\)     | 有向完全图中的任意一个顶点             |
| \(OD(v)\) | 顶点\(v\) 的出度                       |
| \(n\)     | 有向完全图中的顶点个数                 |
| \(n-1\)   | 顶点\(v\) 可以指向除自己外的其他顶点数 |

\[
ID(v)+OD(v)=2(n-1)
\]

**变量说明：**


| 变量       | 含义                       |
| ---------- | -------------------------- |
| \(v\)      | 有向完全图中的任意一个顶点 |
| \(ID(v)\)  | 顶点\(v\) 的入度           |
| \(OD(v)\)  | 顶点\(v\) 的出度           |
| \(n\)      | 有向完全图中的顶点个数     |
| \(2(n-1)\) | 顶点\(v\) 的总度数         |

---

# 七、有向图的入度和出度

## 1. 弧的表示

有向图中，一条弧可以写成：

\[
u\to v
\]

**变量说明：**


| 变量       | 含义                                |
| ---------- | ----------------------------------- |
| \(u\)      | 弧的起点，也叫弧尾                  |
| \(v\)      | 弧的终点，也叫弧头                  |
| \(u\to v\) | 从顶点\(u\) 指向顶点 \(v\) 的有向边 |

---

## 2. 有向图入度和公式

\[
\sum_{i=1}^{n} ID(v_i)=e
\]

**变量说明：**


| 变量                      | 含义               |
| ------------------------- | ------------------ |
| \(n\)                     | 有向图中的顶点个数 |
| \(v_i\)                   | 第\(i\) 个顶点     |
| \(ID(v_i)\)               | 顶点\(v_i\) 的入度 |
| \(e\)                     | 有向图中的弧数     |
| \(\sum_{i=1}^{n}ID(v_i)\) | 所有顶点入度之和   |

---

## 3. 有向图出度和公式

\[
\sum_{i=1}^{n} OD(v_i)=e
\]

**变量说明：**


| 变量                      | 含义               |
| ------------------------- | ------------------ |
| \(n\)                     | 有向图中的顶点个数 |
| \(v_i\)                   | 第\(i\) 个顶点     |
| \(OD(v_i)\)               | 顶点\(v_i\) 的出度 |
| \(e\)                     | 有向图中的弧数     |
| \(\sum_{i=1}^{n}OD(v_i)\) | 所有顶点出度之和   |

---

## 4. 入度和等于出度和

\[
\sum_{i=1}^{n} ID(v_i)=\sum_{i=1}^{n} OD(v_i)=e
\]

**变量说明：**


| 变量             | 含义               |
| ---------------- | ------------------ |
| \(n\)            | 有向图中的顶点个数 |
| \(v_i\)          | 第\(i\) 个顶点     |
| \(ID(v_i)\)      | 顶点\(v_i\) 的入度 |
| \(OD(v_i)\)      | 顶点\(v_i\) 的出度 |
| \(e\)            | 有向图中的弧数     |
| \(\sum ID(v_i)\) | 所有顶点入度之和   |
| \(\sum OD(v_i)\) | 所有顶点出度之和   |

---

## 5. 有向图总度数和

\[
\sum_{i=1}^{n} [ID(v_i)+OD(v_i)]=2e
\]

**变量说明：**


| 变量        | 含义               |
| ----------- | ------------------ |
| \(n\)       | 有向图中的顶点个数 |
| \(v_i\)     | 第\(i\) 个顶点     |
| \(ID(v_i)\) | 顶点\(v_i\) 的入度 |
| \(OD(v_i)\) | 顶点\(v_i\) 的出度 |
| \(e\)       | 有向图中的弧数     |
| \(2e\)      | 所有顶点总度数之和 |

---

# 八、强连通有向图

## 1. 强连通的可达关系

有向图中，任意两个顶点 \(u,v\)，都满足：

\[
u\to v
\]

**变量说明：**


| 变量       | 含义                                     |
| ---------- | ---------------------------------------- |
| \(u\)      | 有向图中的任意一个顶点                   |
| \(v\)      | 有向图中另一个任意顶点                   |
| \(u\to v\) | 从顶点\(u\) 可以沿有向路径到达顶点 \(v\) |

并且：

\[
v\to u
\]

**变量说明：**


| 变量       | 含义                                     |
| ---------- | ---------------------------------------- |
| \(v\)      | 有向图中的任意一个顶点                   |
| \(u\)      | 有向图中另一个任意顶点                   |
| \(v\to u\) | 从顶点\(v\) 可以沿有向路径到达顶点 \(u\) |

这时称该有向图为**强连通图**。

---

## 2. 强连通有向图最少弧数

如果有向强连通图有 \(n\) 个顶点，并且 \(n\ge 2\)，则最少需要：

\[
n
\]

条弧。

**变量说明：**


| 变量       | 含义                                       |
| ---------- | ------------------------------------------ |
| \(n\)      | 强连通有向图中的顶点个数                   |
| \(n\ge 2\) | 表示至少有两个顶点                         |
| \(n\) 条弧 | 构成一个包含所有顶点的有向环所需的最少弧数 |

---

## 3. 最少强连通结构：有向环

\[
v_1\to v_2\to v_3\to \cdots \to v_n\to v_1
\]

**变量说明：**


| 变量                   | 含义                                   |
| ---------------------- | -------------------------------------- |
| \(v_1,v_2,\ldots,v_n\) | 强连通有向图中的\(n\) 个顶点           |
| \(\to\)                | 有向弧或有向路径方向                   |
| \(v_n\to v_1\)         | 最后一个顶点回到第一个顶点，形成有向环 |

---

## 4. 强连通有向图边数范围

\[
n \le e \le n(n-1)
\]

**变量说明：**


| 变量       | 含义                                   |
| ---------- | -------------------------------------- |
| \(n\)      | 强连通有向图中的顶点个数               |
| \(e\)      | 强连通有向图中的弧数                   |
| \(n\)      | 强连通有向图的最少弧数，对应一个有向环 |
| \(n(n-1)\) | 强连通有向图的最多弧数，对应有向完全图 |

---

## 5. 强连通、弱连通关系


| 关系                       | 是否成立 |
| -------------------------- | -------- |
| 强连通一定弱连通           | 成立     |
| 弱连通一定强连通           | 不成立   |
| 强连通图忽略方向后一定连通 | 成立     |
| 有向图连通不一定强连通     | 成立     |

---

# 九、子图相关概念

设原图为：

\[
G=(V,E)
\]

**变量说明：**


| 变量  | 含义                 |
| ----- | -------------------- |
| \(G\) | 原图                 |
| \(V\) | 原图的顶点集合       |
| \(E\) | 原图的边集合或弧集合 |

子图为：

\[
G'=(V',E')
\]

**变量说明：**


| 变量   | 含义                 |
| ------ | -------------------- |
| \(G'\) | 原图\(G\) 的子图     |
| \(V'\) | 子图的顶点集合       |
| \(E'\) | 子图的边集合或弧集合 |

子图必须满足：

\[
V'\subseteq V
\]

**变量说明：**


| 变量          | 含义                 |
| ------------- | -------------------- |
| \(V'\)        | 子图的顶点集合       |
| \(V\)         | 原图的顶点集合       |
| \(\subseteq\) | 表示“是……的子集” |

并且：

\[
E'\subseteq E
\]

**变量说明：**


| 变量          | 含义                 |
| ------------- | -------------------- |
| \(E'\)        | 子图的边集合或弧集合 |
| \(E\)         | 原图的边集合或弧集合 |
| \(\subseteq\) | 表示“是……的子集” |

---

## 子图分类


| 类型     | 说明                           |
| -------- | ------------------------------ |
| 子图     | 顶点和边都是原图的一部分       |
| 生成子图 | 包含原图所有顶点，只删边不删点 |
| 生成树   | 连通图的极小连通生成子图       |
| 连通分量 | 极大连通子图                   |

---

# 十、边数与连通分量的常考关系

设无向图有 \(n\) 个顶点，\(e\) 条边，\(c\) 个连通分量。

## 1. 最少连通分量下界

一开始 \(n\) 个顶点，如果没有边，就是 \(n\) 个连通分量。
每增加一条边，最多可以让连通分量减少 1。

所以：

\[
c\ge n-e
\]

**变量说明：**


| 变量    | 含义                   |
| ------- | ---------------------- |
| \(c\)   | 无向图中的连通分量个数 |
| \(n\)   | 无向图中的顶点个数     |
| \(e\)   | 无向图中的边数         |
| \(n-e\) | 连通分量数量的一个下界 |

---

## 2. 最少连通分量公式

如果边足够多，最少可以连成一个连通图。

\[
c_{\min}=1
\]

**变量说明：**


| 变量         | 含义                             |
| ------------ | -------------------------------- |
| \(c_{\min}\) | 最少连通分量个数                 |
| \(1\)        | 表示整张图连通，只有一个连通分量 |

综合考虑：

\[
c_{\min}=\max(1,n-e)
\]

**变量说明：**


| 变量            | 含义                                                |
| --------------- | --------------------------------------------------- |
| \(c_{\min}\)    | 最少连通分量个数                                    |
| \(n\)           | 无向图中的顶点个数                                  |
| \(e\)           | 无向图中的边数                                      |
| \(\max(1,n-e)\) | 在\(1\) 和 \(n-e\) 中取较大值，保证连通分量至少为 1 |

---

## 3. 最多连通分量问题的核心不等式

想让连通分量尽可能多，就要把边尽量集中放在少数顶点中，让其他顶点孤立。

\[
\frac{s(s-1)}{2}\ge e
\]

**变量说明：**


| 变量                 | 含义                                               |
| -------------------- | -------------------------------------------------- |
| \(s\)                | 用来集中容纳\(e\) 条边所需的最少顶点数             |
| \(e\)                | 无向图中的边数                                     |
| \(\frac{s(s-1)}{2}\) | \(s\) 个顶点的无向完全图最多能容纳的边数           |
| \(\ge\)              | 表示这些顶点最多容纳的边数必须不少于题目给出的边数 |

---

## 4. 示例：21 条边需要几个顶点容纳

\[
\frac{s(s-1)}{2}\ge 21
\]

**变量说明：**


| 变量                 | 含义                         |
| -------------------- | ---------------------------- |
| \(s\)                | 容纳 21 条边所需的最少顶点数 |
| \(21\)               | 题目给出的边数               |
| \(\frac{s(s-1)}{2}\) | \(s\) 个顶点最多能形成的边数 |

当 \(s=7\) 时：

\[
\frac{7\times 6}{2}=21
\]

**变量说明：**


| 变量   | 含义                                   |
| ------ | -------------------------------------- |
| \(7\)  | 用来集中放置这些边的顶点数             |
| \(6\)  | 除自身外，每个顶点最多连接的其他顶点数 |
| \(21\) | 7 个顶点构成无向完全图时的边数         |

如果总共有 51 个顶点，那么剩下孤立顶点数为：

\[
51-7=44
\]

**变量说明：**


| 变量   | 含义                     |
| ------ | ------------------------ |
| \(51\) | 图中的总顶点数           |
| \(7\)  | 用来容纳 21 条边的顶点数 |
| \(44\) | 剩下的孤立顶点数         |

总连通分量数为：

\[
44+1=45
\]

**变量说明：**


| 变量   | 含义                                       |
| ------ | ------------------------------------------ |
| \(44\) | 孤立顶点数量，每个孤立顶点都是一个连通分量 |
| \(1\)  | 由 7 个顶点和 21 条边组成的那个连通分量    |
| \(45\) | 最多连通分量个数                           |

---

# 十一、判断图是否一定连通

## 1. 有 \(n-1\) 条边，不一定连通

很多人会误以为：

\[
e=n-1
\]

**变量说明：**


| 变量    | 含义                           |
| ------- | ------------------------------ |
| \(e\)   | 无向图中的边数                 |
| \(n\)   | 无向图中的顶点个数             |
| \(n-1\) | 树的边数，也是连通图的最少边数 |

就一定是树，这是错的。
必须还要满足“连通”。

---

## 2. 例子：4 个顶点 3 条边不一定连通

\[
n=4,\quad e=3=n-1
\]

**变量说明：**


| 变量    | 含义                |
| ------- | ------------------- |
| \(n\)   | 图中的顶点个数      |
| \(4\)   | 题目给出的顶点数    |
| \(e\)   | 图中的边数          |
| \(3\)   | 题目给出的边数      |
| \(n-1\) | \(4-1=3\)，树的边数 |

虽然满足 \(e=n-1\)，但如果 3 个顶点形成三角形，另一个顶点孤立，它不是连通图，也不是树。

---

## 3. 连通且有 \(n-1\) 条边，一定是树

\[
\text{连通} + e=n-1 \Rightarrow \text{树}
\]

**变量说明：**


| 变量            | 含义               |
| --------------- | ------------------ |
| \(e\)           | 无向图中的边数     |
| \(n\)           | 无向图中的顶点个数 |
| \(n-1\)         | 树的边数           |
| \(\Rightarrow\) | 表示“可以推出”   |

或者：

\[
\text{无回路} + e=n-1 \Rightarrow \text{树}
\]

**变量说明：**


| 变量            | 含义               |
| --------------- | ------------------ |
| \(e\)           | 无向图中的边数     |
| \(n\)           | 无向图中的顶点个数 |
| \(n-1\)         | 树的边数           |
| \(\Rightarrow\) | 表示“可以推出”   |

---

## 4. 边数少于 \(n-1\)，一定不连通

\[
e<n-1
\]

**变量说明：**


| 变量    | 含义                 |
| ------- | -------------------- |
| \(e\)   | 无向图中的边数       |
| \(n\)   | 无向图中的顶点个数   |
| \(n-1\) | 连通图至少需要的边数 |
| \(<\)   | 表示“少于”         |

如果边数少于 \(n-1\)，那么图一定不连通。

---

## 5. 边数足够多时一定连通

一个 \(n\) 个顶点的无向简单图，如果不连通，最多可以让 \(n-1\) 个顶点构成完全图，另一个顶点孤立。

\[
\frac{(n-1)(n-2)}{2}
\]

**变量说明：**


| 变量                     | 含义                                               |
| ------------------------ | -------------------------------------------------- |
| \(n\)                    | 无向简单图中的顶点个数                             |
| \(n-1\)                  | 除孤立点外，其余顶点个数                           |
| \(n-2\)                  | 在这\(n-1\) 个顶点中，每个顶点最多连接的其他顶点数 |
| \(\frac{(n-1)(n-2)}{2}\) | \(n\) 个顶点的非连通简单图最多边数                 |

所以如果：

\[
e>\frac{(n-1)(n-2)}{2}
\]

**变量说明：**


| 变量                     | 含义                           |
| ------------------------ | ------------------------------ |
| \(e\)                    | 无向简单图中的边数             |
| \(n\)                    | 无向简单图中的顶点个数         |
| \(\frac{(n-1)(n-2)}{2}\) | 非连通简单图最多可能拥有的边数 |
| \(>\)                    | 表示“多于”                   |

那么图一定连通。

---

# 十二、回路与桥

## 1. 回路上的边不是桥

如果一条边在某个回路上，那么删除它之后，图仍然连通。


| 边的类型   | 删除后是否连通 |
| ---------- | -------------- |
| 回路上的边 | 仍然连通       |
| 桥         | 会变成不连通   |

---

## 2. 桥的定义

如果删除某条边后，图的连通分量增加，那么这条边就是桥，也叫割边。

树中的每一条边都是桥。

---

# 十三、邻接矩阵相关知识

## 1. 无向图邻接矩阵对称

\[
A[i][j]=A[j][i]
\]

**变量说明：**


| 变量        | 含义                                |
| ----------- | ----------------------------------- |
| \(A\)       | 图的邻接矩阵                        |
| \(i,j\)     | 顶点编号                            |
| \(A[i][j]\) | 顶点\(v_i\) 到顶点 \(v_j\) 是否有边 |
| \(A[j][i]\) | 顶点\(v_j\) 到顶点 \(v_i\) 是否有边 |

无向图没有方向，所以邻接矩阵关于主对角线对称。

---

## 2. 无向图邻接矩阵求度

\[
d(v_i)=\sum_{j=1}^{n} A[i][j]
\]

**变量说明：**


| 变量                      | 含义                                    |
| ------------------------- | --------------------------------------- |
| \(d(v_i)\)                | 顶点\(v_i\) 的度                        |
| \(v_i\)                   | 第\(i\) 个顶点                          |
| \(A\)                     | 无向图的邻接矩阵                        |
| \(A[i][j]\)               | 顶点\(v_i\) 和顶点 \(v_j\) 之间是否有边 |
| \(n\)                     | 图中的顶点个数                          |
| \(\sum_{j=1}^{n}A[i][j]\) | 邻接矩阵第\(i\) 行元素之和              |

---

## 3. 有向图邻接矩阵求出度

\[
OD(v_i)=\sum_{j=1}^{n} A[i][j]
\]

**变量说明：**


| 变量                      | 含义                                        |
| ------------------------- | ------------------------------------------- |
| \(OD(v_i)\)               | 顶点\(v_i\) 的出度                          |
| \(v_i\)                   | 第\(i\) 个顶点                              |
| \(A\)                     | 有向图的邻接矩阵                            |
| \(A[i][j]\)               | 从顶点\(v_i\) 指向顶点 \(v_j\) 的弧是否存在 |
| \(n\)                     | 图中的顶点个数                              |
| \(\sum_{j=1}^{n}A[i][j]\) | 邻接矩阵第\(i\) 行元素之和                  |

---

## 4. 有向图邻接矩阵求入度

\[
ID(v_i)=\sum_{j=1}^{n} A[j][i]
\]

**变量说明：**


| 变量                      | 含义                                        |
| ------------------------- | ------------------------------------------- |
| \(ID(v_i)\)               | 顶点\(v_i\) 的入度                          |
| \(v_i\)                   | 第\(i\) 个顶点                              |
| \(A\)                     | 有向图的邻接矩阵                            |
| \(A[j][i]\)               | 从顶点\(v_j\) 指向顶点 \(v_i\) 的弧是否存在 |
| \(n\)                     | 图中的顶点个数                              |
| \(\sum_{j=1}^{n}A[j][i]\) | 邻接矩阵第\(i\) 列元素之和                  |

---

# 十四、根据图片题目整理的核心公式表


| 图类型             | 公式                   | 变量说明                                      |
| ------------------ | ---------------------- | --------------------------------------------- |
| 无向图度数和       | \(\sum d(v)=2e\)       | \(d(v)\)：顶点度数；\(e\)：边数               |
| 有向图入度和       | \(\sum ID(v)=e\)       | \(ID(v)\)：顶点入度；\(e\)：弧数              |

| 无向连通图最少边   | \(n-1\)                | \(n\)：顶点数                                 |
| 树的边数           | \(e=n-1\)              | \(e\)：边数；\(n\)：顶点数                    |
| 森林边数           | \(e=n-c\)              | \(e\)：边数；\(n\)：顶点数；\(c\)：连通分量数 |

| 强连通有向图最少弧 | \(e=n\)                | \(e\)：弧数；\(n\)：顶点数，且 \(n\ge2\)      |

# 十五、考试易错点总结


| 易错说法                       | 正确理解                   |
| ------------------------------ | -------------------------- |
| 有\(n-1\) 条边的图一定是树     | 错，还要连通               |
| 连通图一定有\(n-1\) 条边       | 错，连通图至少\(n-1\) 条边 |
| 树一定有\(n-1\) 条边           | 对                         |
| 有向图弱连通就是强连通         | 错                         |
| 强连通一定弱连通               | 对                         |
| 无向图所有度数之和等于边数     | 错，是边数的 2 倍          |
| 有向图入度和等于出度和         | 对                         |
| 删除回路上的边会破坏连通性     | 错，仍连通                 |
| 树中删除任意一条边都会不连通   | 对                         |
| 树中增加任意一条边都会产生回路 | 对                         |
| 连通分量是普通连通子图         | 错，是极大连通子图         |

# 十六、做这类题的思路

## 第一步：看是无向图还是有向图

无向图优先用：

\[
\sum d(v)=2e
\]

**变量说明：**


| 变量     | 含义           |
| -------- | -------------- |
| \(d(v)\) | 顶点\(v\) 的度 |
| \(e\)    | 无向图中的边数 |

有向图优先用：

\[
\sum ID(v)=\sum OD(v)=e
\]

**变量说明：**


| 变量      | 含义             |
| --------- | ---------------- |
| \(ID(v)\) | 顶点\(v\) 的入度 |
| \(OD(v)\) | 顶点\(v\) 的出度 |
| \(e\)     | 有向图中的弧数   |

---

## 第二步：看是否连通

如果题目说连通图，马上想到：

\[
e\ge n-1
\]

**变量说明：**


| 变量  | 含义               |
| ----- | ------------------ |
| \(e\) | 连通图中的边数     |
| \(n\) | 连通图中的顶点个数 |

如果说树，马上想到：

\[
e=n-1
\]

**变量说明：**


| 变量  | 含义           |
| ----- | -------------- |
| \(e\) | 树中的边数     |
| \(n\) | 树中的顶点个数 |

如果说森林，马上想到：

\[
e=n-c
\]

**变量说明：**


| 变量  | 含义                 |
| ----- | -------------------- |
| \(e\) | 森林中的边数         |
| \(n\) | 森林中的顶点个数     |
| \(c\) | 森林中的连通分量个数 |

---

## 第三步：看是否完全图

无向完全图：

\[
e=\frac{n(n-1)}{2}
\]

**变量说明：**


| 变量  | 含义                   |
| ----- | ---------------------- |
| \(e\) | 无向完全图中的边数     |
| \(n\) | 无向完全图中的顶点个数 |

有向完全图：

\[
e=n(n-1)
\]

**变量说明：**


| 变量  | 含义                   |
| ----- | ---------------------- |
| \(e\) | 有向完全图中的弧数     |
| \(n\) | 有向完全图中的顶点个数 |

---

## 第四步：看是否强连通

强连通有向图要求任意两点互相可达。

最少弧数：

\[
e=n
\]

**变量说明：**


| 变量  | 含义                                    |
| ----- | --------------------------------------- |
| \(e\) | 强连通有向图中的弧数                    |
| \(n\) | 强连通有向图中的顶点个数，要求\(n\ge2\) |

---
